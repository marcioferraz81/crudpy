Aula 4 - Configurando o bd e criando o gerenciador de rotas .py

Este arquivo tem por finalidade configurar banco de dados, usuário e senha, além de gerenciar as rotas dos arquivos que serão carregados em base.html

Abaixo o app.py

Explicações 1a parte:
  • é importado o Flask (framework) e o responsável em renderizar as páginas que serão trabalhadas
  • é importado o conector MySQL
  • app = Flask(__name__) -> nosso app "enxerga o framework"
  • são aplicadas as configurações de acesso ao banco, no exemplo estamos acessando o localhost (do USBWebServer), com o usuário "root", senha "usbw" e banco de dados "python"
  • criamos um cursor para acessar o banco de dados e trabalhar com o CRUD (cursor = db.cursor())
Explicações 2a parte:
  • podemos observar as rotas definidas como "/", "/view_user" e "/add_user"
  • a rota "/add_user" receberá método POST de um form HTML para ser disparada
  • cada rota aponta para um método e em cada método disparamos uma ação ao BD
  • Nesta imagem podemos observar que existe a consulta de todos os usuários e a inserção.
Explicações 3 a parte:
  • podemos observar as rotas definidas como "/edit_user" e "/delete_user"
  • ambas as rotas recebem um id
  • No caso de "/edit_user", o id é recebido, é disparada uma consulta por id e depois editado o registro
  • Nesta imagem podemos observar que existe a consulta de todos os usuários e a inserção.
  • if __name__ == '__main__': - indicamos o app.py como arquivo principal
