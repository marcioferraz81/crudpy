Aula 6 - index e view_user

Abaixo iremos criar os arquivos index.html e view_user.html (arquivos que usarão a base.html para funcionar).

index.html

Explicações:
  • {% extends "base.html" %} - aplica como base o arquivo indicado
  • {% block content %} - indica o conteúdo que será carregado em base.html
  • {% endblock %} - encerra as alterações aplicadas pelo Flask (Python)

view_user.html

Explicações:
  • href="{{ url_for('add_user') }}" - link para acesso ao form de cadastro
  • através de {% for user in users %} apresentamos os usuários na tabelas
  • através de {% for user in users %} users é retorno do método def view_user():
  • {{ user[0] }} - acesso o campo 0, ou seja, o id (1 - name e assim por diante)
  • href="{{ url_for('edit_user', id=user[0]) }}" - link para editar com o id
  • href="{{ url_for('delete_user', id=user[0]) }}" - link para excluir com o id
  • {% endfor %} - fim foreach
  • {% endblock %} - fim bloco python
