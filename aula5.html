Aula 5 - Criando o arquivo base.html

Este arquivo tem por finalidade carregar o head/body e receber as outras páginas em seu body (nas outras páginas não iremos criar head e body, e sim apenas o conteúdo - forms e listas)

Abaixo a 1a parte do base.html

Explicações:
  • Estamos utilizando a estrutura do HTML5
  • Estilização do projeto o Bootstrap 4.1.3 (português)
  • Os links estão com href apontando para uma rota. Essas rotas estão configuradas no arquivo app.py (exemplo: {{ url_for('view_user') }}

Abaixo a 2a parte do base.html

Explicações:
  • Os links estão com href apontando para uma rota. Essas rotas estão configuradas no arquivo app.py (exemplo: {{ url_for('view_user') }}
  • No div.container temos "block content", local que receberá o conteúdo das outras páginas
